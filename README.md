# البرنامج الزمني الداخلي للاتحاد السعودي للسهام

هذه نسخة كاملة جاهزة لاستبدال ملفات مشروع التقويم الداخلي الحالي مع الاحتفاظ بنفس رابط Vercel.

## الفعاليات المستبعدة

تم استبعاد الفعاليتين التاليتين فقط:

- بطولة القطاع العسكري
- معسكر المنتخب التمهيدي

## إجمالي الفعاليات

يحتوي المشروع على 22 فعالية داخلية.

## الملفات التي يجب أن تظهر في جذر المستودع

- `app`
- `public`
- `package.json`
- `README.md`
- `.gitignore`

## طريقة الاستبدال

1. افتح مستودع `saaf-calendar-v2`.
2. احذف مجلدي `app` و`public` والملفين `package.json` و`README.md` الحاليين.
3. فك ضغط هذا الملف على جهازك.
4. ارفع محتويات المجلد الناتج إلى نفس مستودع GitHub.
5. اضغط `Commit changes`.
6. انتظر Vercel حتى تصبح الحالة `Ready`.

سيبقى رابط الموقع نفسه:

`https://saaf-calendar-v2.vercel.app/`

ورابط تحميل التقويم:

`https://saaf-calendar-v2.vercel.app/api/calendar`
